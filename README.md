# Emirates Expeditions 🌍✈️

A modern travel and tourism website built to showcase premium expedition experiences across the UAE and beyond.
Designed with a clean UI, immersive visuals, and smooth user experience to help travelers discover unforgettable adventures.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/SirLooseNuts/emirates-expeditions.git
```

Navigate into the project folder:

```bash
cd emirates-expeditions
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
